Place your certificate documents (PDFs or images) in this folder.

To link a certificate to the portfolio:
1. Save the file here (e.g. "google_cloud.pdf").
2. Open the main file "script.js" in the root directory.
3. Locate the "certDetails" object.
4. Replace the '#' placeholder link for the corresponding certificate with the path of the file you just saved (e.g., 'assets/certificates/google_cloud.pdf') or a verification link (e.g., Coursera or Forage share link).
5. Refresh the website in your browser!
